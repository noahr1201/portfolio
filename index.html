Level Level Showdown

Level Level Showdown

  • Level Level Showdown
  • September 2024
  • Stage
  • Individueel (Met hulp van collega's)

Dit betreft mijn project genaamd "Level Level Showdown. Ik heb dit project gemaakt als onderdeel van mijn stage bij Level Level. LL specialiseert in toegankelijke wordpress websites. Om bekend te worden met hoe WP ontwikkeling werkt kreeg ik de opdracht om een website te maken voor het halloween personeelsfeest. Ik heb dit gedaan met een webpack, tailwind, twig, npm omgeving. Ik heb de plug-ins gravity forms en ACF blocks gebruikt voor het opzetten van de FAQ blokken en het inschrijf formulier. Mijn obstakels bij dat project waren vooral het opzetten van een wordpress project. En daarnaast de styling met tailwind, ik heb nog niet eerder tailwind gebruikt dus dat was iets nieuws. Ik heb dit project ervaren als een leuk project en ik heb veel geleerd over wordpress ontwikkeling dankzij dit project.

De website wordt gehost op een server van het stagebedrijf, en ik weet daarom niet hoe lang het project open blijft staan. Ik heb daarom verschillende foto's van de website toegevoegd als link zodat je toch een indruk krijgt van wat het eindresultaat is.
